Common Issues
  • Slow startup: Profile with zsh -i -c 'zprof'
  • Key bindings not working: Check for conflicting plugins
  • Tmux sessions not persisting: Verify tmux-resurrect plugin
  • Neovim LSP issues: Check :LspInfo and :checkhealth
  • Git integration problems: Verify git configuration
Quick Fixes
  • Reset configuration: mv ~/.zshrc ~/.zshrc.bak
  • Clear completion cache: rm -rf ~/.zcompdump*
  • Restart tmux server: tmux kill-server
  • Rebuild neovim config: :PackerSync
  • Check health: :checkhealth
