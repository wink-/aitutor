What is Tmux?

Tmux (Terminal Multiplexer) is a powerful tool that allows you to create and manage multiple terminal sessions within a single window. It's like having multiple tabs and split screens in your terminal, but much more powerful.

Why Use Tmux?
  • Persist sessions across SSH disconnections
  • Organize multiple projects in separate sessions
  • Split terminal into multiple panes
  • Share sessions with teammates
  • Create reproducible development environments
  • Background process management
Benefits for AI Development
  • Monitor training logs in real-time
  • Run multiple experiments simultaneously
  • Keep Jupyter notebooks running
  • Manage GPU processes efficiently
  • Quick context switching between projects
  • Remote model serving and monitoring
Real-World AI Development Scenario

Imagine you're training a deep learning model remotely. With tmux, you can:

  • Session 1: Training script running for hours
  • Session 2: Jupyter notebook for data analysis
  • Session 3: TensorBoard monitoring
  • Detach and go home while training continues
  • Reattach from anywhere to check progress
  • Share sessions with team members

Core Concepts

Tmux has a hierarchical structure with three main components. Understanding this hierarchy is crucial for mastering tmux.

Tmux Hierarchy

Session: "ai-project"

A persistent workspace containing multiple windows

Window 1: "editor"

Pane 1: Neovim

Pane 2: Terminal
Window 2: "training"

Pane 1: Training Script

Pane 2: TensorBoard
Window 3: "jupyter"

Pane 1: Jupyter Lab
Sessions

Independent workspaces that persist even when detached. Perfect for organizing different projects or environments.

  • Can be named for easy identification
  • Persist after disconnection
  • Can be shared between users
  • Maintain state across reboots
Windows

Similar to tabs in a browser. Each window can have a different layout and purpose within a session.

  • Numbered and named
  • Quick switching with shortcuts
  • Independent layouts
  • Status bar indicators
Panes

Split views within a window. Each pane runs its own shell instance independently.

  • Horizontal/vertical splits
  • Resizable boundaries
  • Focus switching
  • Zoom functionality

Essential Keybindings

Tmux uses a prefix key (default: Ctrl+b) followed by a command key. Master these shortcuts for efficient workflow.

Important: All keybindings below assume you press and release Ctrl+b first, then press the command key.

Command Keybinding Description
Start session tmux new -s name Create a new named session
List sessions Ctrl+b s Interactive session selector
Detach session Ctrl+b d Detach from current session
Attach session tmux attach -t name Reattach to named session
Rename session Ctrl+b $ Rename current session
Kill session tmux kill-session -t name Terminate named session

Command Keybinding Description
New window Ctrl+b c Create a new window
Next window Ctrl+b n Move to next window
Previous window Ctrl+b p Move to previous window
Select window Ctrl+b 0-9 Jump to window by number
Last window Ctrl+b l Switch to last active window
Rename window Ctrl+b , Rename current window
Kill window Ctrl+b & Close current window
List windows Ctrl+b w Interactive window selector

Command Keybinding Description
Split vertical Ctrl+b % Split pane left/right
Split horizontal Ctrl+b " Split pane top/bottom
Navigate panes Ctrl+b ←→↑↓ Move between panes
Next pane Ctrl+b o Cycle through panes
Toggle zoom Ctrl+b z Zoom/unzoom current pane
Resize pane Ctrl+b Ctrl+←→↑↓ Resize in direction
Close pane Ctrl+b x Kill current pane
Show pane numbers Ctrl+b q Display pane numbers

Command Keybinding Description
Enter copy mode Ctrl+b [ Enter copy/scroll mode
Exit copy mode Esc or q Exit copy mode
Start selection Space Begin text selection
Copy selection Enter Copy selected text
Paste Ctrl+b ] Paste copied text
Search up ? Search backwards in copy mode
Search down / Search forwards in copy mode
Pro Tips:
  • Press Ctrl+b ? to see all available keybindings
  • Use Ctrl+b : to enter command mode for advanced operations
  • Hold Ctrl+b and tap q to quickly show pane numbers

Configuration

Customize tmux with a configuration file at ~/.tmux.conf to boost your productivity.

Basic Configuration

# ~/.tmux.conf - Basic Configuration # Change prefix from Ctrl+b to Ctrl+a (easier to reach) unbind C-b set-option -g prefix C-a bind-key C-a send-prefix # Enable mouse support set -g mouse on # Start windows and panes at 1, not 0 set -g base-index 1 setw -g pane-base-index 1 # Easier pane splitting bind | split-window -h -c "#{pane_current_path}" bind - split-window -v -c "#{pane_current_path}" # Vim-style pane navigation bind h select-pane -L bind j select-pane -D bind k select-pane -U bind l select-pane -R # Vim-style window navigation bind -n M-h previous-window bind -n M-l next-window # Reload config file bind r source-file ~/.tmux.conf \; display "Config reloaded!" # Increase history limit set-option -g history-limit 50000 # Improve colors set -g default-terminal "screen-256color" set-option -sa terminal-overrides ',xterm-256color:RGB'

AI Development Configuration

# ~/.tmux.conf - AI Development Enhanced # Status bar customization set -g status-bg colour235 set -g status-fg white set -g status-left-length 40 set -g status-left '#[fg=green]#S #[fg=blue]#I:#P #[default]' set -g status-right '#[fg=cyan]%Y-%m-%d %H:%M #[fg=yellow]#(uptime | cut -d "," -f 1)' # Window status format setw -g window-status-current-format '#[fg=white,bg=red] #I:#W ' setw -g window-status-format '#[fg=cyan,bg=black] #I:#W ' # Pane border colors set -g pane-border-style fg=colour238 set -g pane-active-border-style fg=colour39 # Message colors set -g message-style bg=colour235,fg=colour166 # Copy mode improvements setw -g mode-keys vi bind-key -T copy-mode-vi 'v' send -X begin-selection bind-key -T copy-mode-vi 'y' send -X copy-selection-and-cancel # Quick pane cycling unbind ^A bind ^A select-pane -t :.+ # Synchronize panes (useful for cluster operations) bind y setw synchronize-panes # Monitor activity in other windows setw -g monitor-activity on set -g visual-activity on # Better session management bind S command-prompt -p "New session:" "new-session -A -s '%%'" bind K confirm-before -p "Kill session #S? (y/n)" kill-session # Resize panes more easily bind -r H resize-pane -L 5 bind -r J resize-pane -D 5 bind -r K resize-pane -U 5 bind -r L resize-pane -R 5

Theme Configuration

# ~/.tmux.conf - Custom Theme # Tokyo Night Theme set -g status-style bg='#1a1b26',fg='#a9b1d6' set -g window-status-current-style bg='#7aa2f7',fg='#1a1b26' set -g pane-border-style fg='#3b4261' set -g pane-active-border-style fg='#7aa2f7' set -g message-style bg='#1a1b26',fg='#7aa2f7' # Custom status bar with system info set -g status-left '#[fg=#7aa2f7,bg=#1a1b26] #S #[fg=#a9b1d6,bg=#1a1b26] ' set -g status-right '#[fg=#9ece6a]CPU: #{cpu_percentage} #[fg=#e0af68]MEM: #{ram_percentage} #[fg=#bb9af7]%H:%M ' # Window list format setw -g window-status-format ' #I:#W ' setw -g window-status-current-format ' #I:#W ' # Plugins (requires TPM - Tmux Plugin Manager) set -g @plugin 'tmux-plugins/tpm' set -g @plugin 'tmux-plugins/tmux-sensible' set -g @plugin 'tmux-plugins/tmux-resurrect' set -g @plugin 'tmux-plugins/tmux-cpu' # Initialize TMUX plugin manager (keep at bottom) run '~/.tmux/plugins/tpm/tpm'
Installation Instructions
  1. Download your preferred configuration file
  2. Save it as ~/.tmux.conf in your home directory
  3. Reload tmux: tmux source-file ~/.tmux.conf
  4. Or restart tmux to apply changes

Common Troubleshooting

Solutions to common tmux issues you might encounter.

Mouse scrolling doesn't work

Problem: Mouse wheel scrolling doesn't work in tmux

Solution: Add mouse support to your configuration:

# Add to ~/.tmux.conf set -g mouse on # Reload configuration $ tmux source-file ~/.tmux.conf
Colors look wrong in vim/neovim

Problem: Syntax highlighting colors are incorrect in editors

Solution: Fix terminal color support:

# Add to ~/.tmux.conf set -g default-terminal "screen-256color" set-option -sa terminal-overrides ',xterm-256color:RGB' # For older tmux versions set-option -sa terminal-overrides ',xterm-256color:Tc'
Can't copy/paste with system clipboard

Problem: Copy/paste doesn't work between tmux and system clipboard

Solution: Configure clipboard integration:

# macOS $ brew install reattach-to-user-namespace # Add to ~/.tmux.conf set-option -g default-command "reattach-to-user-namespace -l $SHELL" # Linux with xclip $ sudo apt install xclip # Add to ~/.tmux.conf bind -T copy-mode-vi y send-keys -X copy-pipe-and-cancel 'xclip -in -selection clipboard'
SSH connection lost, how to recover session?

Problem: Lost SSH connection and need to recover work

Solution: Sessions persist automatically:

# SSH back to the server $ ssh user@server # List all running sessions $ tmux ls # Attach to your session $ tmux attach -t session-name # Or attach to the most recent session $ tmux attach
Prefix key conflicts with other applications

Problem: Ctrl+b conflicts with other keybindings

Solution: Change the prefix key:

# Change to Ctrl+a (popular alternative) unbind C-b set -g prefix C-a bind C-a send-prefix # Or use Ctrl+Space unbind C-b set -g prefix C-Space bind C-Space send-prefix
Tmux starts slowly

Problem: Tmux takes a long time to start

Solution: Optimize your configuration:

# Remove or optimize slow plugins # Check for expensive status bar updates # Reduce history limit if set too high set-option -g history-limit 10000 # Profile your configuration $ tmux -f /dev/null -L test new-session -d 'sleep 1' $ tmux -L test list-sessions $ tmux -L test kill-server
"No such session" error

Problem: Can't attach to a session that should exist

Solution: Check session status and names:

# List all sessions $ tmux ls # If no sessions exist, create one $ tmux new -s main # Check for detached sessions $ tmux list-sessions # Kill problematic sessions $ tmux kill-session -t session-name
